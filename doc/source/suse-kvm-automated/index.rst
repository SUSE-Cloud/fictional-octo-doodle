.. _kvm-automated-deploymentguide::

====================================
SUSE KVM Automated (Developers Only)
====================================

This guide provides instructions for performing a Containerized Openstack
Cloud deployment on a developer workstation. It bootstraps
a minimal SUSE CaaSP cluster with SES rook and a deployer VM.

This feature is experimental and is intended for SUSE developers.
